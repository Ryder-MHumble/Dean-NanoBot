# Sentiment Report 质检门禁

在发送日报前，必须逐项检查并全部通过：

1. 维度完整：包含“官方账号运营分析 + 全网舆情洞察”两大部分（若有数据）。
2. 优先级完整：重点条目都带 `P1/P2/P3`（高风险至少 `P1/P2`）。
3. 链接完整：风险、机会、高互动样本都附原始链接；若缺失要显式标注。
4. 行动可执行：执行清单必须是可落地动作，不是空泛建议。
5. 长度受控：
   - `Standard`（默认）：总长度建议 `<= 8000` 中文字符。
   - `Fast`（按需）：总长度建议 `<= 5500` 中文字符。
6. 不落地文件：默认直接回复或发送消息，不创建本地 `.md` 报告文件。

可使用 `scripts/validate_intel_report.py` 做快速机器校验：

```bash
python3 scripts/validate_intel_report.py --max-chars 8000 --file - <<< "$report_content"
```
