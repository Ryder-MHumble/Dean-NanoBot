# 舆情分析指导原则

本文档提供舆情分析的最佳实践、方法论和注意事项，用于指导准确的情感分类和风险评估。

## 情感分类标准

### 正面情感 (Positive)

**定义**: 表达积极、赞赏、满意的情感

**关键指标**:
- 包含正面关键词: 优秀、很好、专业、领先、创新、推荐、值得、满意、赞等
- 表达喜爱、支持、认可的态度
- 分享积极的体验和结果
- 鼓励他人参与或尝试

**典型表达**:
- "这个研究院真的很专业，老师都很负责"
- "课程质量很高，学到了很多东西"
- "强烈推荐，值得来学习"
- "研究成果领先，创新能力很强"

**注意事项**:
- 避免将客观描述误判为正面（如"今天去了研究院"）
- 注意识别讽刺和反语（中文语境中较少但需警惕）

### 中性情感 (Neutral)

**定义**: 客观陈述、无明显情感倾向

**关键指标**:
- 纯粹的事实陈述
- 提问但无明显倾向
- 信息分享无评价
- 对比分析但不下结论

**典型表达**:
- "今天参加了研究院的开放日"
- "有人了解这个研究院的招生政策吗？"
- "研究院位于中关村，主要研究AI"
- "我在考虑三个研究院，包括这个"

**注意事项**:
- 中性不代表不重要，某些中性内容可能隐含关注点
- 大量针对特定话题的中性询问可能预示潜在问题
- 对比类内容需要仔细判断是否含有倾向性

### 负面情感 (Negative)

**定义**: 表达不满、批评、失望、担忧的情感

**关键指标**:
- 包含负面关键词: 问题、担心、失望、不满、投诉、质疑、差、坑等
- 表达批评、抱怨、警告的态度
- 分享负面体验
- 不推荐或劝阻他人

**典型表达**:
- "感觉有点失望，和预期不符"
- "课程质量有问题，老师不够专业"
- "不太推荐，性价比不高"
- "担心研究成果的实用性"

**注意事项**:
- 建设性批评也属于负面，但严重程度较轻
- 需要区分个人情绪发泄和实质性问题
- 高互动的负面内容风险更大

## 中文语境特殊考虑

### 1. 委婉表达

中文表达常常比较含蓄，需要理解言外之意：

- "还行吧" = 一般般，可能偏负面
- "emmm" = 犹豫、不确定，可能有保留意见
- "怎么说呢" = 通常后面跟负面评价
- "不是很..." = 否定，负面
- "有待提高" = 委婉的批评

### 2. 网络用语

熟悉常见网络表达：

**正面**:
- "绝了", "YYDS"（永远的神）, "爱了爱了", "OMG", "太可了"
- "牛", "强", "6666", "厉害了"

**负面**:
- "翻车", "踩雷", "坑", "割韭菜"
- "emmm", "呵呵"（讽刺）
- "也就那样", "不过如此"

**中性**:
- "吃瓜", "围观", "路过"

### 3. 表情符号

表情符号能显著影响情感：

**正面**: 😊 😄 👍 ❤️ 🌟 ✨ 💯 🔥
**负面**: 😔 😤 😠 💔 👎
**中性**: 🤔 😐 🤷

## 风险评估框架

### 风险等级判定

#### 🔴 高优先级风险

**判定标准**:
1. 包含严重负面关键词：投诉、举报、欺骗、骗局、违规、违法
2. 明确指向实质性问题：安全、财务、法律、诚信
3. 高互动量（意味着广泛传播）
4. 来自有影响力的账号

**行动要求**: 24小时内响应

**示例**:
- "感觉被骗了，准备投诉"
- "课程质量严重不符，要求退款"
- "涉嫌虚假宣传"

#### 🟡 中优先级风险

**判定标准**:
1. 包含一般负面关键词：问题、不满、失望
2. 指向服务或体验问题
3. 中等互动量
4. 可能引发更多讨论

**行动要求**: 一周内处理

**示例**:
- "课程安排有点混乱"
- "老师回复不够及时"
- "感觉性价比不高"

#### ⚪ 低优先级/监控

**判定标准**:
1. 温和的负面表达或建设性批评
2. 低互动量
3. 个人主观感受
4. 不太可能引发连锁反应

**行动要求**: 持续监控

**示例**:
- "个人觉得进度有点快"
- "希望能增加实践环节"
- "还可以，但有改进空间"

### 风险升级机制

以下情况需要提升风险等级：

1. **快速传播**: 短时间内大量转发/评论
2. **媒体关注**: 被媒体或大V转发
3. **群体效应**: 多人表达类似问题
4. **情绪升级**: 讨论从中性转向负面
5. **外部联动**: 与其他负面事件关联

## 平台特性分析

### 小红书 (Xiaohongshu)

**用户特征**:
- 年轻女性为主（18-35岁）
- 注重生活品质和体验
- 分享欲强，重视社区互动

**内容特点**:
- 图文为主，视觉呈现重要
- 真实体验分享
- 详细的评测和对比
- 标签系统发达

**舆情特点**:
- 正面：详细的好评，"种草"推荐
- 负面：详细的"避雷"贴，"拔草"警告
- 影响力：口碑效应明显，决策影响大

**监控重点**:
- 体验类内容（课程、环境、服务）
- 对比评测（与其他机构）
- 招生相关咨询和讨论

### 抖音 (Douyin)

**用户特征**:
- 年龄跨度大
- 娱乐导向
- 快节奏消费内容

**内容特点**:
- 短视频为主（15-60秒）
- 视听冲击力强
- 病毒式传播快
- 评论区活跃

**舆情特点**:
- 正面：点赞快速增长，"网红"效应
- 负面：负面内容传播极快
- 影响力：爆发式传播，覆盖面广

**监控重点**:
- 视频内容的情感基调
- 评论区的讨论方向
- 高播放量内容（>10K）

### B站 (Bilibili)

**用户特征**:
- 年轻人群（18-30岁）
- 二次元文化
- 高学历、技术宅
- 重视深度内容

**内容特点**:
- 长视频为主
- 专业性强
- UP主（创作者）影响力大
- 弹幕文化

**舆情特点**:
- 正面：专业认可，"三连"支持
- 负面：技术性批评，"鬼畜"讽刺
- 影响力：社区凝聚力强

**监控重点**:
- 技术和专业内容评价
- 会议/讲座录播反馈
- UP主的观点和评论

### 微博 (Weibo)

**用户特征**:
- 广泛年龄层
- 关注时事和热点
- 意见领袖影响大

**内容特点**:
- 短文本为主
- 快速传播
- 话题标签系统
- 转发链强大

**舆情特点**:
- 正面：正能量传播
- 负面：容易形成舆论风暴
- 影响力：最具公共舆论属性

**监控重点**:
- 话题标签讨论
- 大V转发和评论
- 突发事件和危机

## 热点话题分析方法

### 识别真热点 vs 假热点

**真热点特征**:
- 自然增长的讨论量
- 多平台同时出现
- 不同角度的讨论
- 持续时间较长（>3天）

**假热点特征**:
- 突然爆发但快速消退
- 单一平台或账号推动
- 讨论角度单一
- 明显的营销痕迹

### 话题生命周期

1. **萌芽期**: 少量讨论，需要密切观察
2. **成长期**: 讨论量快速增长，介入最佳时机
3. **爆发期**: 话题达到峰值，需要主动管理
4. **衰退期**: 讨论量下降，总结和归档

## KOL 识别和管理

### KOL 分类

**头部 KOL**:
- 粉丝 > 100万
- 影响力巨大
- 合作成本高
- 风险和机会并存

**腰部 KOL**:
- 粉丝 1万-100万
- 性价比高
- 互动率好
- 合作性强

**长尾 KOL**:
- 粉丝 < 1万
- 垂直领域
- 真实用户
- 口碑价值高

### KOL 评估指标

1. **影响力**: 粉丝数、认证状态
2. **活跃度**: 发布频率、更新规律
3. **互动率**: 赞评比、转发率
4. **内容质量**: 专业性、原创性
5. **情感倾向**: 历史内容的态度

### KOL 管理策略

**正面 KOL**:
- 建立长期关系
- 提供独家内容
- 邀请深度合作

**中性 KOL**:
- 保持沟通
- 提供真实信息
- 争取转化为正面

**负面 KOL**:
- 了解负面原因
- 私下沟通解决
- 避免公开冲突

## 常见误判案例

### 案例 1: 情感表达的复杂性

❌ **错误判断**: "这个研究院，emmmm，怎么说呢" → 中性
✅ **正确判断**: 负面（"emmm"和"怎么说呢"暗示保留意见）

### 案例 2: 反讽和讽刺

❌ **错误判断**: "呵呵，真是好得很呢" → 正面
✅ **正确判断**: 负面（"呵呵"在中文网络语境中常表示讽刺）

### 案例 3: 建设性批评

❌ **错误判断**: "整体不错，但希望能增加实践环节" → 负面
✅ **正确判断**: 正面偏中性（虽有建议但整体肯定）

### 案例 4: 询问类内容

❌ **错误判断**: "有人知道这个研究院靠谱吗？" → 负面
✅ **正确判断**: 中性（纯粹询问，但需关注后续讨论）
