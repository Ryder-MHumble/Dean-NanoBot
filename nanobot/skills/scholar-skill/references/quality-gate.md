# Scholar 输出质检门禁

在最终回复前，必须逐项检查并全部通过：

1. 结构完整：包含查询结论、学者条目（可选统计和导师学生关系）、必要时能力边界说明。
2. 优先级完整：所有展示学者都带 `P1/P2/P3`，按 `P1 -> P2 -> P3` 排序。
3. 链接完整：
   - 优先使用 `profile_url` 并标注原始链接。
   - 若缺失，明确标注“链接缺失”，必要时给内部记录链接。
4. 消歧透明：同名或低置信结果不可强行定人，必须给候选与歧义说明。
5. 不落地文件：默认直接回复，不创建本地 `.md`/`.markdown` 报告文件。
6. 长度受控：
   - Standard 模式（默认）：`P1<=6`、`P2<=4`、`P3<=2`。
   - Fast 模式（按需）：`P1<=3`、`P2<=2`、`P3<=1`。

若任一项不满足，先修正再发送。
